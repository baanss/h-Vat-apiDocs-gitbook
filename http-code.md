# HTTP Code

현재 이용중인 HTTP CODE 에 대한 내용입니다.\
(아래의 내용은 현재 수정중입니다. 추가되거나 변경될 수 있습니다)

| Code | Message      | Description           |
| ---- | ------------ | --------------------- |
| 200  | Success      |                       |
| 400  | Bad Request  |                       |
| 401  | Unauthorized | 로그인 불가                |
| 403  | Forbidden    | 로그인. but 데이터 접근 권한 없음 |
| 500  | Server Error |                       |
